void GridEnvironment::calculate_TZ ( )
static

calculate the potential and realized transition zones

GridEnvironment::calculate_TZ: Calculates the potential and realized transition zone cells in the landscape STEP 1: The function goes through all grid cells, asks whether the cell is part of an arable patch and, if so, whether it has a forest or grassland cell next to it. If so, it is set to a potential transition zone cell and the number of border cells is stored for the specific patch. STEP 2: The total number of realized transition zones are determined. Then the arable patches are ranked/prioritized by the area. Either by decreasing or ascending order, cells are marked as transition zones: A random cell is chosen, if the cell belongs the current patch and is a potential transition zone cell, it is marked as realized transition zone cell and the number of 'to be realized' transition zone cells are decreased. The resulting grid is saved for later analyses. (one only considering the TZ and one considering the land use classes.
